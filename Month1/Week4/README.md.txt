# Week 4 – AWS Automation with Python (boto3)

This week focuses on using Python and boto3 to interact with AWS services such as S3 and EC2.

## What I Learned
- How to configure AWS CLI and IAM credentials
- Using boto3 to interact with AWS
- How to list S3 buckets programmatically
- How to list EC2 instances programmatically

## Scripts
- list_s3_buckets.py – Lists all S3 buckets in your AWS account
- list_ec2_instances.py – Lists EC2 instances and states
